Etiquetas HTML y Etiquetas


HTML es un markuo language, esto quiere decir que está escrito con código que se puede leer si que sea necesario en compilarlo primero, esto quiere decir que el texto de una página web este marcado que da instrucciones al navegador web como se tiene que mostrar el texto, estas son considerado como etiquetas de marcado de HTML. Estas escribiendo etiquetas HTML, esta hechas con un numero de partes específicas como: el carácter “menor que <”, el carácter “mayor que >”, etc.

El hipertexto (HTML) este es un lenguaje informático que esta conformadas las paginas web y plataformas online, este no se puede considerar un lenguaje de programación, esta no pude crear funciones dinámicas, los usuarios web se crea la estructura sección, párrafo y enlaces que están conformados por elementos, etiquetas y atributos, están conformadas 142 etiquetas HTML disponible.

Elementos HTML

Los elementos HTML se dividen de tres partes, con la combinación de estas tres se crea un elemento HTML:

  • Etiqueta de apertura: Esta cataloga o indica donde comienza el elemento, que se encuentran en corchetes de apertura y cierre para crear un párrafo.

  • Contenido: es el contenido que ve el espectador.

  • Etiqueta de cierre: Este es misma etiqueta de apertura con la diferencia con una barra inclinada delante del nombre del elemento

    para culminar el párrafo.

    • Atributos HTML

    Fundamental de los elementos HTML son los atributos y cuentan con dos secciones:

  • Nombre: Esta es la identifica la información adicional que quiere agregar.

  • Valor del atributo: esta da mas detalles que el anterior.
